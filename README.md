🧠 NeuroCortex V12 Apex

NeuroCortex V12 Apex is a cognitive neuro-dynamic simulation system inspired by biological processes, thermodynamics, and adaptive behavior models.
It simulates mental energy, allostatic load, and synaptic plasticity in real time using an asynchronous Python architecture.

✨ Key Features
🧠 Dynamic cognitive states (ZENITH, FLOW, STABLE, FRAGILE, BURNOUT)
⚡ Homeostatic energy system (0 → 100 scale)
🧪 Allostatic load simulation (cumulative stress modeling)
🔁 Adaptive synaptic plasticity (stimulus reinforcement & decay)
📊 Neuro-dynamic engine with exponential fatigue penalties
💾 Atomic JSON persistence system
🧵 Async-safe architecture (asyncio + locks)
🧾 Real-time CLI logging with visual indicators
